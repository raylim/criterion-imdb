Criterion IMDb Overlay Privacy Policy

Last updated: June 4, 2026

Criterion IMDb Overlay is a Firefox extension that shows IMDb ratings and related film metadata on Criterion pages.

What data the extension stores

  • Your extension settings, such as display preferences and optional OMDb API keys.
  • Locally cached score lookup results used to speed up repeated page loads.
  • A locally cached copy of the hosted Criterion score index fetched from this project’s GitHub Pages site.

What data the extension sends

The extension does not send your browsing history or account data to a private backend operated by this project.

When score data is missing from the bundled or hosted cache, the extension may request public film metadata from third-party services needed to resolve a title:

  • https://raylim.github.io/criterion-imdb/ for the hosted cache file
  • https://raw.githubusercontent.com/ as a fallback hosted cache source
  • https://www.omdbapi.com/ if you enable OMDb fallback with your own API key
  • https://v2.sg.media-imdb.com/ and https://v3-cinemeta.strem.io/ for public title matching fallback

What data is not collected

  • No user accounts are created for this extension.
  • No personal profile is built or sold.
  • No analytics, advertising IDs, or tracking pixels are included.

Control

You can remove stored settings and local cache by clearing the extension’s stored data or uninstalling the extension.
